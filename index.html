[02] ABOUT

Not just tickets.
Access.

SceneKyaHai is a brutalist movie-ticket booking system built as a DBMS demonstration — a fully normalized SQL backend (MySQL) powering a stripped-down front-end. No clutter. No dark patterns. Just select, sit, and stare at the screen.

Every booking is an atomic transaction — seat collisions are rejected at the database layer via a stored procedure, not just in the UI. The seat map you see is a live VIEW on top of the bookings table.
